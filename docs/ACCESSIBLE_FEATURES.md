# Seoul Accessible Transit - 장애인 친화 대중교통

## 🎯 프로젝트 비전

**"모든 사람이 안전하고 편리하게 대중교통을 이용할 수 있도록"**

서울의 장애인, 노약자, 유모차 이용자를 위한 접근 가능한 대중교통 경로 안내 서비스

---

## 🔑 핵심 기능

### **1. 무장애 경로 찾기**
- ✅ 엘리베이터만 이용하는 경로
- ✅ 계단 없는 경로 우선
- ✅ 휠체어 접근 가능 경로만 표시
- ✅ 경사로 정보 포함

### **2. 역사 접근성 정보**
- 🚇 엘리베이터 위치 및 개수
- 🚪 휠체어 리프트 유무
- 🚻 장애인 화장실 위치
- 🅿️ 장애인 주차장 정보
- 📏 플랫폼-열차 틈새 정보

### **3. 저상버스 실시간 정보**
- 🚌 저상버스 도착 시간
- 🚌 일반/저상버스 구분 표시
- 🚌 휠체어 탑승 가능 버스만 필터링

### **4. 실시간 시설 상태**
- ⚠️ 엘리베이터 고장 알림
- ⚠️ 에스컬레이터 점검 정보
- ⚠️ 공사 구간 우회 안내

### **5. 음성 안내 (접근성)**
- 🔊 TTS 음성 경로 안내
- 🔊 화면 읽기 지원
- 🔊 큰 글씨 모드

---

## 📊 데이터 소스

### **공공 데이터**
1. **서울 열린데이터 광장**
   - 지하철 무장애 편의시설 정보
   - 장애인 화장실 위치
   - 엘리베이터 정보

2. **서울시 교통정보 시스템**
   - 저상버스 운행 정보
   - 버스 도착 정보 API

3. **국토교통부 장애인 편의시설 정보**
   - 역사별 접근성 등급
   - 휠체어 리프트 설치 현황

### **크라우드소싱**
- 사용자 제보 (엘리베이터 고장, 경로 정보)
- 커뮤니티 피드백

## 🎨 UI/UX 특화

### **접근성 우선 설계**
```
홈 화면
  ├─ 큰 버튼 (터치 영역 넓게)
  ├─ 고대비 모드
  ├─ 음성 입력 지원
  └─ 간단한 인터페이스

경로 옵션
  ├─ [기본] 가장 빠른 경로
  ├─ [무장애] 엘리베이터만 이용
  ├─ [저상버스] 저상버스 우선
  └─ [안전] 계단 없는 경로

경로 상세
  ├─ 각 구간별 접근성 정보
  ├─ 엘리베이터 위치 지도
  ├─ 예상 이동 시간 (휠체어 기준)
  └─ 대체 경로 (엘리베이터 고장 시)
```

### **Phase 3D: UI 개선 (1주)**
- [ ] 접근성 옵션 추가

### **Phase 3E: 테스트 및 피드백 (1주)**

- [ ] 피드백 수렴
